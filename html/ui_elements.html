Buttons

Primary Button

This button should be used as default whenever there is something to confirm, or for a more specific interaction, like connecting the Spotify account or returning from the playlist view.

HTML


                        <button class="btn">button</button>

CSS(SCSS)


                        .btn{
                            color: inherit;
                            background-color: $color-orange;
                            border: 0.1rem solid $color-orange;
                            border-radius: 1.7rem;
                            padding: .5rem 2.8rem;
                            cursor: pointer;
                            transition: all .2s ease-in-out;
                            font-family: inherit;
                            font-size: 17pt;
                            
                            &:hover{
                                color: $color-orange;
                                background-color: transparent;
                            }
                        }

Navigation

The navigation consists of a hamburger menu that extends to a full screen navigation menu. It is designed that way, because there should be a little disappearance as possible between the mobile- and desktop version. This approach works on both sizes well, because it draws the attention to the navigation options. For an example see top right

Form Elements

We chose a free, but still structured approach to forms. That’s why the forms consist of open input fields and round buttons (see Buttons), that are vertically aligned.

Das is a Form

or

HTML


                        <div class="forms">
                        <h2>Das is a Form</h2>
                        <label for="inputUsr" class="formlabel">Username</label>
                        <input id="inputUsr" class="input" type="text"     placeholder="Username">
                        
                        <label for="inputPw" class="formlabel">Password</label>
                        <input id="inputPw" class="input" type="password" placeholder="Password">
                        
                        <div class="submitBtns">
                        <input type="submit" class="btn" value="Login">
                        <p>or</p>
                        <input type="submit" class="btn btn--second" value="Create Account" formaction="/elsewhere">
                        </div>
                        </div>

                        .forms{
